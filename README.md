

### Installation and usage

1.  
   ```
   git clone git@github.com:adhinneupane/brbytes.git
   ```
    
2. Make sure you have nodejs and express installed
   ```
   node --version
   npm install express 
   ```

3.  
   ```
   cd ICT-blockly-P5/blocklyp5-node/ &&
   node app.js
   ```

4. 
   ```
   visit server on port 8080/home
   ```
